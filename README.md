# BioQuantix AI Technologies: Core PoC & Framework

This repository contains the official Proof-of-Concept (PoC) code and theoretical architecture for BioQuantix AI Technologies. We implement a hybrid computing framework that couples Neural Ordinary Differential Equations (Neural ODEs) with Variational Quantum Eigensolvers (VQE) to overcome the static limitations of legacy structural biology.

## 🧬 Project Overview
* Target Molecule: Human Insulin Protein (PDB ID: `1znj`)
* Temporal Dynamics: Continuous-time atomic trajectory simulation via torchdiffeq.
* Quantum Energy Estimation: Molecular binding energy calculation via PennyLane VQE.
* Core Documentation: Theoretical framework and mathematical derivations are fully detailed in our internal BioQuantix Technical Whitepaper.
