# BioQuantix AI Technologies: Core PoC

This repository contains the official Proof-of-Concept (PoC) code for BioQuantix AI Technologies, implementing a hybrid computing architecture that couples Neural Ordinary Differential Equations (Neural ODEs) with Variational Quantum Eigensolvers (VQE) to solve the static limitations of legacy structural biology.

## Project Overview
* Target Molecule: Human Insulin Protein (PDB ID: `1znj`)
* Temporal Dynamics: Continuous-time atomic trajectory simulation via torchdiffeq.
* Quantum Energy Estimation: Molecular binding energy calculation via PennyLane VQE.
